# imageboard
Тестовый проект имиджборды.

# Установка
Для установки надо создать виртуальное окружение python в любой папке. Затем в эту папку скопировать данный репозиторий, и запустить команду (Windows): `venv\Scripts\activate` Данная команда активирует виртуальное окружение. Затем надо установить все необходимые пакеты командой `pip install -r requirements.txt`

# Создание БД
Для создания БД необходимо запустить команду `flask db init` Для создания сценария миграции и его применения надо выполнить последовательно следующие команды: `flask db migrate` и `flask db upgrade`

# Запуск приложения
Для запуска приложения надо выполнить команду `flask run`


UPD: текущие миграции БД не подходят для sqlite, чтобы развернуть локально надо пуллить без папки migrations.
